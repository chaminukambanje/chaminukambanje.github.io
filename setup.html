Module 1 Overview

Set Up Business Central

Goal: Initialize the environment, migrate data, and secure the system.

1.1 Create and Configure a New Company

Focuses heavily on RapidStart Services (Configuration Packages). You must know how to map Excel columns to BC Table Fields and validate data before applying.

1.2 Manage Security

Security is handled via Permission Sets and User Groups. Understand the difference between assigning permissions directly vs. using groups for efficiency.

1.3 Core Settings

Distinguish between global settings (General Ledger Setup) and user-specific exceptions (User Setup).
