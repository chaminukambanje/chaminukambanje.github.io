Module 1

Set Up Business Central

1.1 Create and Configure a New Company

This section focuses on the RapidStart Services (Configuration Packages).

  • Configuration Packages: Used to import data from Excel. You must map the Excel columns to the Table Fields.
  • Validate Package: Checks data for errors before importing.
  • Apply Package: Commits the data to the database.

1.2 Manage Security

Security is handled via Permission Sets.

  • User Groups: A bundle of permission sets (e.g., "D365 ACCOUNTANTS"). Assigning a user to a group is faster than assigning individual permissions.
  • Effective Permissions: A page where you can see exactly what a specific user can do (Read, Insert, Modify, Delete).

1.3 Core Settings

User Setup vs. General Ledger Setup:

  • General Ledger Setup: Sets the global "Allow Posting From/To" dates.
  • User Setup: Overrides the global dates for specific users (e.g., allowing the admin to post in a closed month).
